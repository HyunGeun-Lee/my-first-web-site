기본지식

html : 태그와 태그사이에 언어를 넣어 뼈대를 만드는데 사용
css : 색칠, 꾸미기 가능 (예: 흰색 초록색 배경회색 버튼초록색 등 )
js : ( 자바스크립트 , 로직이 있음) 복잡한 로직 설정 아이디 미입력표시 로그인 실패시 등등

태그

  • a : 링크
  • U (underline) : 밑줄긋기
  • STRONG : 글씨강조 (굵게)
  • p : 문단을 나타냄 ( 여백이 정해져 있음)
  • div : 구역을 나눠줌
  • br : 줄을 바꾸는 태그 (종료태그가 없다)
  • h1~6 : 제목을 나타내는 태그
  • img : 이미지 삽입태그 ( 종료태그가 없다.)
  • alt : 대체 텍스트,이미지를 보여주지 못할떄
  • width,height : 이미지 가로세로 크기지정
  • hr : 수평선
  • ol : 순서있는 리스트
  • ul : 순서없는 리스트
  • li : 리스트
  • table : 표
  • tr : 줄
  • td : 칸(셀)
  • th : 제목 셀
  • li : 목를 나타냄
